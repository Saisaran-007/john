# John v1.5

John v1.5 is a clean, practical assistant UI built as a thinking partner.

## Features

- Dashboard
- Chat with reasoning
- Visible memory page
- Goal tracking
- Constitution page
- Voice selection and speech output
- Automatic memory capture
- Transparent "Why?" reasoning panel
- Optional OpenAI-compatible API endpoint

## How to use

1. Upload these files to your GitHub repository.
2. Enable GitHub Pages on the `main` branch with the root folder.
3. Open the live site on your Chromebook or Android phone.

## Notes

- The assistant works with a local fallback brain even without an API key.
- You can later connect an OpenAI-compatible endpoint in Settings.
- Voice quality depends on the voices available on your device/browser.
